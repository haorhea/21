<p>他算得上早熟，但对性却并不了解，也不能怪他。其他孩子的启蒙多数从一个问题开始：我是从哪儿来的？保守些的家长支吾其词，编些白鹳衔来的啊，包心菜里蹦出来的啊，诸如此类的不靠谱瞎话来搪塞，开明些的父母则用更适合儿童的语言来向孩子说明，生命的诞生与起源大致是怎样一回事。而对于花京院来说，这两个选项都是不存在的。他还在孤儿院的时候，连父母都没有，问老师我是怎么来的呀，老师便答，你是你爸爸妈妈生的，于是问题的重点就转向了“我爸爸妈妈去了哪儿”；至于被领养后，这个问题就更加不在日程表上了，问承太郎这种问题太自讨没趣，反正他知道自己不是承太郎生的就是，其他的管他这么多呢。</p> 
<p>因此到了那堂雄蕊雌蕊，遮遮掩掩的保健体育课，花京院依旧是个懵懵懂懂的十四岁少年，他知道世上存在性行为这档子事，但具体操作如何，行为后果如何，不了解也不关心。这堂课过去了，他依旧混沌着，班里的同学却不是这样。同一个年龄段里，总是有那么几个孩子在这方面早早开了窍，姑娘们相对矜持，男生却多半乐于炫耀，那堂课像是开闸泄洪，原本独自藏着掖着的东西，突然不再像原来那样私密。一个月之内，男同学之间就开始私下交流起一些已然几经转手的东西来，大家偷偷摸摸，心照不宣，直到那盘录像带送到花京院手里，而他困惑地问：这是什么？</p> 
<p>他不是那种只知道死读书的所谓优等生，成绩顶尖，但平时和班里的其他人也够打成一片，因此问出这种问题来，也没人笑他。录像带的来源，那个满下巴痘痘的男生告诉他，是好东西，回去趁你爸妈不在家的时候看，不用谢。</p> 
<p>承太郎经常加班，这天打电话回来说让他先吃晚饭，花京院就便热饭菜边把录像带推进了机子，事后想想，实在不应该在吃饭的时候看这东西。正片开始的时候，他吓了一跳，筷子落在地上，啪嗒一下，于是又吓了第二跳，明知家里只有他一个人，还是哆哆嗦嗦地死命按遥控器，猛然关了电视。</p> 
<p>他耳朵里嗡嗡地响，耳尖烫得仿佛要融化，心跳飞快，像是要挣出胸膛。这样难受了二十来分钟，晚饭也冷了，纠结再三，最后又满怀罪恶感地打开电视，把那卷成人录影带给看完了，小电影结束之后立刻跳起来，把带子抽出机器，奔进房间藏进书包的最底层。他心里说不上来是兴奋还是害怕，亦或是有些恶心，脑子里一团乱麻，下身有了反应，又胀又痛，很难受。花京院学着刚才录影带里的男人所做的一样，躲进厕所进行了人生中的第一次自慰，那感觉并不很好，与其说是自我慰藉，不如说一切都只是为了最后释放的一瞬间。射出来之后，承太郎正巧回来了，开门声，招呼声，脚步声，一下一下近了，最后停在卫生间门外，隔着一层毛玻璃问怎么了，是不是肚子不舒服？</p> 
<p>花京院觉得整个人都快烧着了，几乎是带着哭腔，歇斯底里地答道，没有！我好得很！你别进来！</p> 
<p>自此之后，花京院典明大概算是开了窍。他不和人谈论这方面的话题，同学们谈起，他不参与，但是会听；偶尔自慰，只是自慰对他来说并不如其他男生们所形容的那样美妙，确切来说，就仿佛跑了一场马拉松，冲过终点的时候，欢呼和成就感的确令人欣喜，然而他并不想天天都为了那点欣喜就去跑马拉松，漫长的摩擦和触摸伴随着一定快感，可是很乏味，几乎毫无乐趣可言。</p> 
<p>他从来没想过和承太郎交流这方面的内容，而承太郎自然也什么都不知道。</p> 
<p>&nbsp;</p> 
<p>国中的最后一次家长会，老师千叮咛万嘱咐，很重要，很重要，务必请同学们与家长一同出席。他于是提前让承太郎把时间空出来，下午来参加他的家长会。当天早上要去上学之前，犹豫再三，最后还是说了：</p> 
<p>“承太郎，下午到我学校的时候，别穿那件白衣服了吧？”</p> 
<p>“怎么了？”</p> 
<p>“穿那件，大家都看着你……”</p> 
<p>承太郎苦笑一下，说我以前穿黑的，你老说找不到我，找不到的时候还要哭呢，现在反而嫌弃起来了。花京院用筷子戳着荷包蛋，看着蛋黄一点点流出来，不太高兴地说，别老把我当小孩儿。</p> 
<p>他有点烦躁，一方面是因为被当做小孩，另一方面是因为承太郎误解了他的意思，他并不是觉得承太郎穿着那件白风衣不伦不类丢人现眼什么的，只是再好看，这衣服也太招摇太惹眼，承太郎原本就是容易吸引眼球的人，穿上这件之后直接变成了扎眼，而他不喜欢学校里的其他人被他的监护人所吸引，好好的自己的东西，被人觊觎的不安感当然让他不爽。</p> 
<p>“那我穿什么过去呢？”</p> 
<p>“就普通的，挑件普通的西装什么的吧。”</p> 
<p>承太郎点头答应，到了下午一点，如约而至，也如约穿了件普通的西装，换了普通的皮鞋，甚至把几乎从不离身的帽子也给摘了，头发向后梳，用了发胶。花京院在学校走廊里看到他的时候，心里咯噔一下。监护人走过来揉揉他的脑袋，说是先去老师办公室还是先去你教室啊？</p> 
<p>花京院说，教室，接着就一扭头，撂下承太郎，往前一个人走了。</p> 
<p>承太郎跟着他到了教室，在他的座位旁边坐下，果不其然，女同学，男同学，男女同学的妈妈们，几乎个个都扭头过来看着他。花京院坐在座位上，假装在做一张卷子，心里其实很想跳上桌子大叫一声：都把头回过去，我的，不准看。</p> 
<p>承太郎穿西装非常好看，他穿什么都好看，只是因为平时很少穿西装，因此难得穿上的时候，这种好看变得能一下击中人的心脏，而与他朝夕共处了十年的花京院就这样被击中了。他以前就不喜欢别人围在他的承太郎身边，心里总是觉得这种事会发生的原因是那些人讨厌，那些人不好，那些人谄媚，总之都是那些人的错，今天却猛然发现承太郎也是要为这事负些责任的，他怎么能这么，这么好看呢？花京院偷偷从卷子的边缘用余光瞟着承太郎，他几乎都后悔让他今天换西装过来了。</p> 
<p>家长会进行得很顺利，花京院成绩好，性格也没什么缺陷，不惹麻烦，在会上被表扬了几次。看得出他被表扬，承太郎很开心，他习惯成自然地在大家为被表扬的同学鼓掌的时候有点儿害羞地伸手去压了压帽檐，只是今天没戴帽子，于是又讪讪地把手收回去，转而对花京院笑了笑。</p> 
<p>家长会的事原本就算过去了，顶多让花京院意识到，承太郎穿西装有多么好看，而他有多么不喜欢别人盯着他的监护人，这似乎并没有什么重要的意义，和原来没什么两样。然而家长会过了一周之后，某个百无聊赖的夜晚，还有两个月就要十五岁的花京院躺在床上心想，今晚要不撸一下？他的手摸到了自己，熟练地蹭了两下，感觉到了熟悉的些微快感，一切似乎都和往常差不多，直到有一个画面猛然窜进他的脑海，而他的身体因为这个画面而产生了前所未有的巨大反应。</p> 
<p>那个画面是穿着西装的承太郎。</p> 
<p>在做这种事的时候想到自己的监护人，一阵负罪感条件反射地袭上心头，然而罪恶感只有一瞬间，随着想象而来的快感却维持了几秒，后者成功地击败了理智，纵容他再次尝试，以承太郎为性幻想对象。他小心翼翼地闭上眼睛，深呼吸，在脑海中勾勒出承太郎那天的样子：西装，没有帽子，背头，长长的腿，肩很宽，但腰线收进去，坐在他的座位旁边，扭过头对他笑。</p>

<p>为什么这个幻想会是是承太郎？</p> 
<p>但是在考虑清楚这个问题之前，他终于像他的朋友，那些普通的国中男生一样，开始体会到了自慰的乐趣，并由于初尝甜头而刹不住车，当晚消耗了小半盒抽纸，以至于第二天早上精神不振，浑浑噩噩，承太郎都问他是不是身体不舒服，怎么有了黑眼圈。</p> 
<p>他眼神游移不敢回答，甚至不敢看一眼监护人的脸，只能盯着他撑在餐桌上的手，手指修长，骨节分明，形状和大小都非常漂亮，他收回视线，盯着自己眼前的盘子不再出声，而当晚承太郎的手也被归纳入库，成为了他性幻想中的一部分，在夜晚那些光怪陆离的幻觉里对他做了很多不能说出来的事。</p> 
<p>这种感情的变质过于突如其来，也过于猛烈，国中生花京院典明不能很好的应对。十年前他被空条承太郎从孤儿院里接出来没几天的时候，他就决定了他喜欢承太郎，这种喜欢维持到今日，原本不足为奇。他喜欢承太郎，因为他对他很好，他领养了他，他照顾他，他了解他，然而这种喜欢却突然扯上了性，上述冠冕堂皇的理由就都不能成立了，这是一种截然不同的喜欢。</p> 
<p>而花京院甚至不知道这种不同究竟是什么时候开始的。</p>

<p>花京院伴随着困惑和迷恋，就这样上了高中，原本的不安逐渐转变为焦躁，因为他对承太郎的喜欢已经和原来不一样了，他得寸进尺，承太郎却还在原地踏步。花京院站在车站前的书店翻过《如何攻略年长的TA》之类的书，书里说他要利用自己的优势，不要试图装成熟，该怎样就是怎样，用自己年轻又活泼的一面去征服那个年长的对象。</p> 
<p>狗屁，他放下书的时候心想，他已经一直都把我当小孩了，我不可能用这个征服他。</p> 
<p>这本书不适合他的情况，但书店里也没有《如何攻略年长的监护人》这样分门别类到针对个人情况的著作，于是败兴而归，纠结了一阵，饥不择食慌不择路，竟然写邮件去问号称恋爱大师的波鲁那雷夫咨询。他们虽然感情好，但有些事就算是好友也不能随便说，花京院于是在通信邮件里将自己写得像个单纯为恋爱所恼的少年，一切都停留在柏拉图层面，丝毫不提及自己隔着浴室门偷听承太郎洗澡的声音，还闻他换下来没有洗过的睡衣T恤之类的事，自慰什么的更加三缄其口从来没有说。然而遮遮掩掩的咨询起了反效果，向来喜欢煽风点火的波鲁那雷夫大概真把他当做了什么纯情少年，竟然劝他顺其自然。</p> 
<p>怎么可能顺其自然，你还是不是法国人了。</p>

<p>争吵的后果是监护人提高了出差频率，承太郎很受伤，他说“这样你就不必成天看见我”的时候，眉头嘴角都耷拉下来，花京院看着也不怎么高兴。但是出差也有好处，承太郎走了之后，花京院入侵了他的房间，开他的衣柜，睡他的床。躺在承太郎床上的时候，也不敢造次，顶多把脸埋进枕头，嗅那股子夹杂着海水烟草味道的淡淡气息罢了。花京院像个还留存了一丁点儿理智的瘾君子，他离不开承太郎的气息，但是成瘾的同时又要小心盘算，一次用量不能太多，以免这次爽过了头，胃口越来越大，以后就再也不能满足了。</p> 
<p>他最后一次这么干的时候，恰巧被承太郎撞见，监护人提早到家，开门就看到花京院躺在他床上。他没有起疑心，没有拒绝花京院赖在他床上的要求，甚至回应了他的告白，但是什么都没有变，所有的一切都是基于和原先一样的感情基础所得来的，他爱花京院，像一个真正的父亲爱他的孩子。</p> 
<p>但花京院不是什么孩子。还有五十天他就十六岁了。</p> 
<p>他心想，生日那天，我会让你知道的，我已经不是小孩子了。</p>
